# projetPerso

Introduction :

Ce site permet d'afficher les caractéristiques de plusieurs voiture en fonction de la marque.
l'utilisateur choisit une marque depuis le menu déroulant et les caractéristiques s'affichent.
il peut aussi ajouter une voiture et ses caractéristiques dont la marque s'affichera dans le menu déroulant.
Enfin, il pourra effectuer une recherche par id d'une voiture. 
si celle-ci n'existe pas dans la base de donnée, il y aura une redirection vers une page d'erreur.

liste des endpoints :

http://localhost:8080/garage/cars

http://localhost:8080/garage/newCar?marque=audi&modele1=A3&modele2=A4&modele3=A6&color=bleu

http://localhost:8080/garage/id
